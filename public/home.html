Project Notice

Our project is a donation platform designed to connect donors with recipients in need of essential items such as food, clothes, and other necessities. The system allows recipients to enter their details, including name, address, and preferred donation category, and matches them with nearby donors using data stored in a MongoDB database. The frontend is built using AJAX and jQuery for dynamic updates, while cookies are used to manage user sessions and preferences. The backend, powered by Node.js and Express.js, ensures efficient data handling and seamless communication. By leveraging modern web technologies, this platform simplifies the donation process, promotes resource sharing, and fosters community support.
